# CSV Studio

Lightweight CSV Editor with SQL Query Support

A modern desktop application built with Wails, providing a smooth CSV data editing experience with powerful SQL query capabilities.


## Powerful Features
Everything you need to work with CSV files efficiently

- CSV Editing:
Double-click to edit cells with support for multi-line text. Insert, delete rows and columns with ease.

- SQL Query:
Filter and sort your data using familiar SQL syntax. Supports SELECT, WHERE, ORDER BY and more.

- Data Export：
Export your data to Excel (.xlsx) with formatting or JSON format for further processing.

- Undo/Redo：
Full undo/redo support with keyboard shortcuts (Ctrl+Z, Ctrl+Y). Never lose your work again.

- High Performance：
Virtual scrolling technology ensures smooth performance even with large datasets.

- Cross-Platform:
Built with Wails for native performance on Windows, macOS, and other platforms.
